# PROYECTO 1: TARJETA DE PERFIL PERSONAL
## Objetivo
Crear una tarjeta de presentación personal utilizando HTML y CSS que demuestre comprensión de la estructura básica de una página web y estilos fundamentales.

## Descripción
Desarrollarás una tarjeta de perfil estilo "business card" digital. La tarjeta debe presentar información personal de manera organizada y visualmente atractiva. Deberás crear dos archivos: un archivo HTML para la estructura del contenido y un archivo CSS para los estilos visuales.

## Requerimientos Funcionales

*La tarjeta debe incluir:*

- Una fotografía o avatar (puede ser una imagen de placeholder)
- Nombre completo como título principal
- Título profesional o rol (ejemplo: "Desarrollador Web", "Estudiante de Ingeniería")
- Descripción breve personal (2-3 líneas)
- Mínimo 3 enlaces a redes sociales o contacto (GitHub, LinkedIn, Email, etc.)

### Requerimientos Técnicos
1. HTML

- Estructura HTML5 válida con DOCTYPE, head y body
- Meta tags básicos (charset, viewport)
- Título de página apropiado
- Uso de etiquetas semánticas cuando sea apropiado
- Imagen con atributo alt descriptivo
- Enlaces con atributo href funcional

2. CSS

- Archivo CSS externo vinculado correctamente
- Definición de fuente personalizada (puede usar Google Fonts)
- Esquema de colores coherente (mínimo 3 colores)
- La tarjeta debe estar centrada en la pantalla
- Aplicación de margin y padding apropiados
- Border-radius para esquinas redondeadas
- Efecto hover en los enlaces
- Ancho máximo de la tarjeta: 400px

## Criterios de Evaluación

- **Estructura HTML (30%):** Código semántico, bien indentado y organizado
- **Estilos CSS (30%):** Aplicación correcta de propiedades, consistencia visual
- **Diseño Visual (20%):** Atractivo estético, legibilidad, uso apropiado de espacios
- **Funcionalidad (20%):** Todos los enlaces funcionan, imagen se carga correctamente

### Entregables

1. Archivo index.html
2. Archivo styles.css
3. Carpeta images con la foto de perfil
4. Archivo README.md explicando brevemente el proyecto

### Tiempo Estimado:
1-2 días

### Recursos Sugeridos

1. Google Fonts: https://fonts.google.com
2. Placeholder de imágenes: https://placeholder.com o https://unsplash.com
3. Paletas de colores: https://coolors.co

## Notas Adicionales

- No uses frameworks CSS como Bootstrap o Tailwind
- Todo el CSS debe ser escrito por ti
- Asegúrate de que tu código esté bien comentado
- Prueba tu página en al menos dos navegadores diferentes
